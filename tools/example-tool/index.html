📝 示例工具

这是一个示例工具模板，展示如何创建新的网页工具

如何使用这个工具？

这是示例工具的主体内容。你可以在这里放置任何你的工具功能。

新增工具流程

如果想创建新工具，按以下步骤操作：

  • 在 tools/ 目录下创建新的文件夹，例如 tools/my-tool/
  • 在文件夹中创建 index.html 文件（或任意名字的 HTML 文件）
  • 在 HTML 中添加 <title> 标签作为工具名称
  • 在 HTML 中添加 <meta name="description" content="..."> 作为工具描述
  • GitHub Actions 会自动扫描并生成 tools.json
  • 推送到 main 分支后，工具就会出现在首页

HTML 元标签示例

<!DOCTYPE html> <html> <head> <meta charset="UTF-8"> <title>我的工具</title> <meta name="description" content="这是我的工具描述"> </head> <body> <!-- 你的内容 --> </body> </html>

注意事项

  • 工具 HTML 文件可以叫任何名字，例如 index.html, tool.html 等
  • 如果没有 <title> 标签，系统会使用文件夹名作为工具名称
  • 如果没有 description meta 标签，工具卡片不会显示描述
  • 工具可以包含 CSS、JavaScript、图片等其他文件
  • 相对路径会自动处理（基于 /tools/你的工具名/）

后续步骤

你可以：

  • 复制此目录作为新工具的模板
  • 修改 <title> 和 description 元标签
  • 将你的工具代码放入 HTML 中
  • 提交代码到仓库
